# netty-chatroom
一个用netty搭建的web聊天室，可实现群聊和私聊。jdk最低版本要求1.8

# 搭建步骤

1.下载项目到本地

2.导入项目到ide

3.运行ApplicationMain.java启动netty服务

4.浏览器打开chat-server/src/main/webapp/html/index.html文件进入登录界面，多开tab可模拟多个用户。
